CYCO MIDI Channels

Previous( Local configuration )     Next( MIDI Controllers )     Home     Contents     

Where possible CYCO uses standard MIDI channel numbers 1 through 16 inclusive. Some low-level functions however use the actual binary values between 0 and 15. To reduce ambiguity the unqualified term "channel" always refers to MIDI channel numbers 1 through 16. The term "channel-index" (or at times "cindex") indicates a binary value between 0 and 15.

CYCO channels may be assigned symbolic names using the ASSIGN-CHANNEL! function. A symbolic channel name may refer to another symbolic name, as long as circular assignments are avoided. When CYCO encounters a symbolic channel name it looks up the names value recursively until it obtains an actual numeric channel.

            (assign-channel!  alias  value)

The file Examples/local/local-settings.lisp contains examples for using assign-channel!.
