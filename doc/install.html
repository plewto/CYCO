CYCO Installation

Previous( Home )     Next( Project )     Home     Contents     

CYCO is implemented in pure Lisp and any standard Common Lisp should work. It has been tested against both Armed Bear (ABCL) and Steel Banks (SBCL). Steel Banks is generally faster, but in the future interaction with Java applications may be added, which would point towards Armed Bear as the preferred Lisp.

Installation is straight forward. On startup CYCO reads a configuration file used to adapt CYCO to the local studio environment. There is no pre determined structure for the configuration file. It is simply a Lisp file, which may load additional files, and may be structured in any way that makes local sense. However the file does need to exists, even if it does nothing.

The Examples/local folder contains the configuraiton files used for my personal studio and may be used as a guide. In the mean time create the following empty file ~/.config/cyco/local-settings.lisp

            
            cd ~
            # If ~/.config does not exists, create it
            mkdir .config
            mkdir .config/cyco
            touch .config/cyco/local-settings.lisp
Otherwise there is not much to do.
  1. Place CYCO directory wherever you like.
  2. Navigate to the CYCO folder.
  3. Fire up your favorite Lisp.
  4. At the Lisp prompt enter (load "src/cyco")
  5. After CYCO loads enter (cyco)

As for what to do next, read the manual.
