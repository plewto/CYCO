CYCO Amplitude Values

Previous( Metric Values )     Next( Timesig )     Home     Contents     

CYCO amplitudes are real numbers between 0.0 and 1.0. At project render time amplitudes are converted to MIDI velocity 0..127. Thus amplitude has a resolution of 1/127 or ~0.007874. Amplitude values of 0 are considered as rest and all negative amplitudes are "promoted" to 0. All amplitudes greater then 1.0 are clipped to 1.0.

Symbolic amplitudes borrow from the common music notation of P and F. The following amplitude symbols are defined:

            
            R    -   0/127  (rest)
            PPPP -   6/127
            PPP  -  19/127
            PP   -  32/127
            P    -  45/127
            MP   -  58/127
            MF   -  71/127
            F    -  84/127
            FF   -  97/127
            FFF  - 110/127
            FFFF - 123/127

With exception to R, the symbols above may be append with either - or + for finer grained values.

            
            PPPP- -   2/127
            PP+   -  36/127
            P-    -  40/127
            P     -  45/127
            P+    -  49/127
            MP-   -  53/127
            FFFF+ - 127/127

Amplitude related functions:

            
            (AMPLITUDE-P obj)
            
            Predicate test if obj is a valid amplitude.
            
            
            (AMPLITUDE obj &key (range '(0.0 . 1.0)))
            
            Convert obj to real amplitude value. Values outside of range are clipped.
            
            AMPLITUDE is defined for float, symbol and list.  Unrecognized symbols 
            are not converted and used as the return value.
            
            (AMPLITUDE -9)    -> 0.0
            (AMPLITUDE  0)    -> 0.0
            (AMPLITUDE 0.5)   -> 0.5
            (AMPLITUDE 'R)    -> 0.0
            (AMPLITUDE 'PP)   -> 0.252
            (AMPLITUDE 'PIG)  -> PIG
            (AMPLITUDE '(0.5 F MF- PIG)  ->  (0.5 0.661 0.52 PIG)
            
            
            (AMPLITUDE->VELOCITY amp &key (range '(0.0 . 1.0)))
            
            Converts amplitude to MIDI velocity.  Argument may be any type
            accepted by AMPLITUDE.
            
            
            (AMPLITUDE-SYMBOL n)
            
            Returns closest symbolic amplitude for value n.  
            AMPLITUDE-SYMBOL is defined for float and list.
            
            (AMPLITUDE-SYMBOL 0.45)   ->  MP-
            (AMPLITUDE-SYMBOL '(0.1 0.3 0.75))  ->  (PPPP+ PP+ FF-)
