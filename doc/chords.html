CYCO Chord Dictionary

Each PROJECT maintains a chord dictionary for use with EPART. See STRUMMER for an alternate method of producing chords.

*GLOBAL-CHORD-DICTIONARY* is the master chord dictionary. When a project
is created it inherits *GLOBAL-CHORD-DICTIONARY* but may define
project specific chords.

Chord related functions:

            
            (?CHORDS &optional (silent nil))
            
            Displays a list of defined chords for the current project.
            If optional silent flag is t, return the list without printing it.
            
            
            
            (CHORD-DICTIONARY &optional (parent *GLOBAL-CHORD-DICTIONARY*))
            
            Create new CHORD-DICTIONARY with given parent.
            
            
            
            (PARSE-CHORD root cname &key
                                    (project *project*)
                                    (inversion 0)
                                    (octave nil)
                                    (copies nil))
            
            Generate keynumber list using current project chord-dictionary.
            ARGS:
              root       - Keynumber, the chord root.
              cname      - Symbol or list, the chord name or a list of intervals
              :project   - Default *project*
              :inversion - Fixnum, chord inversion 0..mx where mx is the number
                           of notes in the chord. 
              :octave    - Boolean, if true add an octave copy of the initial chord note.
              :copies    - Fixnum or list of fixnums.   For each value, transpose the 
                           base chord by that amount and append to results. Default 0.
            RETURNS: List
            
            
            
            (CHORD! name template &key (project *project*))
            
            Define new chord for current project.
            ARGS:
              name     - Symbol, the chord name. By convention chord names
                         are enclosed in square brackets as in [MAJ].
              template - List of note intervals.  The root key has value 0.
              :project - Defaults to current project.
