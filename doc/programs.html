PROGRAMS is a type of PART used to generate MIDI program changes.

            
            (PROGRAMS name time instruments &key 
                           (qfn #'bar)
                           (project *project*)
                           (section nil)
            
            ARGS:
            name       - Symbol, part name
            time       - Time of event.  Format should be as expected by qfn.
                         For default qfn = #'bar, time is (bar beat subbeat).
            instrments - List of instruments.
            :qfn       - Function. Time cuing function.
            :project   - Default *PROJECT*
            :section   - Default current section of project.
            
            part is bound to variable named name.

Typical usage:

            
            (section verse1)
            
            (programs v1-programs (1 1 1) (list piano organ bass))
            
            ...
            
            (mute v1-programs nil)
            (->midi verse1)
