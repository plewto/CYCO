The CONTROLBALL and related BENDBALL and PRESSUREBALL classes are MIDI controll, bend and pressure versions of QBALL.

They all have similar structures:

            
            (CONTROLBALL name ctrl instruments &key
                                      (cue '())
                                      (values 0)   ;; A pattern as in QBALL
                                      (period nil)
                                      (qfn #'bar)
                                      (project *project*)
                                      (section nil))

Values are "normalized" between 0.0 to 1.0 for controlball and pressureball, -1.0 and +1.0 for bendball.

            
            (PRESSUREBALL name instruments &key
                                      (cue '())
                                      (values 0) 
                                      (period nil)
                                      (qfn #'bar)
                                      (project *project*)
                                      (section nil))
            
            (BENDBALL name instruments &key
                                      (cue '())
                                      (values 0)
                                      (period nil)
                                      (qfn #'bar)
                                      (project *project*)
                                      (section nil))
