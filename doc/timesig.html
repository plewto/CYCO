CYCO Timesig

Previous( Amplitude )     Next( Chords )     Home     Contents     

The CYCO TIMESIG object is a generalization of common notion of a time-signature. TIMESIG maintains the following fields:

  • BARS - Number of bars in a phrase.
  • BEATS - Number of beats per bar.
  • SUBBEATS - Number of subbeats per beat.
  • TSUBBEATS - Number of 'tsubbeats' per beat.
  • UNIT - basic beat unit.
  • TEMPO - tempo in beats per minute.

SUBBEATS is typically used to indicate the number of sixteenth notes per beat. Assuming 4/4 time, SUBBEATS would have a value of 4.

TSUBBEATS is an alternate to SUBBEATS and is typically used for the number of sixteenth note triplets per beat. For 4/4 time. TSUBBEATS is 6.

The possible beat units are: 'W 'H 'Q (default) 'E and 'S

TIMESIG is a sub-class of NODE so that an instance of TIMESIG may have a parent TIMESIG. All null fields in the child are inherited from the parent. The SECTION class is a sub-class of TIMESIG which inherits default values form the parent PROJECT.

TIMESIG related functions:

            
            (TIMESIG &key 
                     (parent *root-timesig*)
                     (name nil)
                     (tempo nil)
                     (unit nil)
                     (bars nil)
                     (beats nil)
                     (subbeats nil)
                     (tsubbeats nil))
            
            Creates new TIMESIG object.
            ARGS:
              parent    - Timesig, default *ROOT-TIMESIG* 
              name      - Symbol, (rarely used)
              tempo     - Float, tempo in Beats per Minute (BPM), 
                          inherited from parent, default 60.
              unit      - Symbol, basic beat unit, may be W H Q E or S, 
                          inherited from parent, default 'Q.
              bars      - Fixnum, number of bars per phrase, 
                          inherited from parent, default 1.
              beats     - Fixnum, number of beat units per bar,
                          inherited from parent, default 4.
              subbeats  - Fixnum, number of subbeats (usually 1/16 note) per beat,
                          inherited from parent, default 4.
              tsubbeats - Fixnum, number of "t"subbeats (usually 1/16 triplet) per beat,
                          inherited from parent, default 6.
            
            (UNIT ts)              Returns basic beat unit.
            (BAR-COUNT ts)         Returns bars per phrase.
            (BEAT-COUNT ts)        Returns beats per bar.
            (SUBBEAT-COUNT ts)     Returns subbeats per beat.
            (TSUBBEAT-COUNT ts)    Returns tsubbeats per beat
            (TICK-COUNT ts)        Returns number of ticks per beat.
            (TEMPO ts)             Returns tempo in BPM.
            (TEMPO! ts bpm)        Changes timesig tempo.
            (DURATION ts)          Returns duration of 1 phrase in seconds.
            (PHRASE-DURATION ts)   Same as DURATION.
            (BAR-DURATION ts)      Returns duration of 1 bar.
            (BEAT-DURATION ts)     Returns duration of a single beat.
            (SUBBEAT-DURATION ts)  Returns duration of a single subbeat.
            (TSUBBEAT-DURATION ts) Returns duration of a single tsubbeat.
            (TICK-DURATION ts)     Returns duration of single tick.
